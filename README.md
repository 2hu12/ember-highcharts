# Ember-highcharts

A [Highcharts](http://www.highcharts.com/) component for [Ember CLI](http://www.ember-cli.com/).

## Installation

* `npm install --save-dev ember-highcharts`
* `ember g ember-highcharts`

## Usage

```handlebars
{{high-charts content=chartData chartOptions=chartOptions}}
```
